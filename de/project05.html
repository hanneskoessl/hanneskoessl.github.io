Entwicklung einer Quiz-Webanwendung mit Django

1. Einleitung

In diesem Projekt habe ich eine Webanwendung zur Erstellung und Durchführung von Quizzen entwickelt. Ziel war es, eine einfache und flexible Plattform zu bauen, auf der Quizze erstellt, bearbeitet und ausgewertet werden können. Dabei lag der Fokus auf einer klaren Backend-Struktur sowie einer übersichtlichen Benutzeroberfläche.

2. Zusammenfassung

Die Anwendung ermöglicht es, Quizze anzulegen, Fragen hinzuzufügen und mehrere Antwortmöglichkeiten pro Frage zu definieren. Fragen können in mehreren Quizzen verwendet werden, wodurch ein wiederverwendbarer Fragenpool entsteht. Nach der Durchführung eines Quiz erhält der Benutzer eine Auswertung der Antworten.

3. Methoden und Tools

Für die Umsetzung wurden folgende Technologien verwendet:

  • Python / Django - Backend, Routing und Datenbankzugriff
  • Django ORM - Modellierung der Datenbank und Beziehungen zwischen Quizzen, Fragen und Antworten
  • HTML / CSS - Struktur und Styling der Benutzeroberfläche
