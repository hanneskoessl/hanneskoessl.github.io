# Portfolio Website - Based on Massively by HTML5 UP

This repository contains the source code for my portfolio website, built using the **Massively** template by HTML5 UP. The design is text-focused and article-oriented, featuring a large background image, smooth parallax effects, and scroll animations powered by Scrollex.

---

## **About Massively**
Massively is a free template created by HTML5 UP. It provides a clean and modern layout that is perfect for showcasing articles, projects, or any content-heavy website.

Original template URL: [HTML5 UP - Massively](https://html5up.net/massively)  
License: [Creative Commons Attribution 3.0](https://html5up.net/license)

---

## **Modifications**
I have customized the template to create a personal portfolio site. Changes include:
- Updated content to reflect my professional and personal projects.
- Adjusted styling.
- Enhanced functionality with additional features.
