Development of a Quiz Web Application with Django

1. Introduction

In this project, I developed a web application for creating and conducting quizzes. The goal was to build a simple and flexible platform where quizzes can be created, edited, and evaluated. The focus was on a clear backend structure as well as a user-friendly interface.

2. Summary

The application allows users to create quizzes, add questions, and define multiple answer options per question. Questions can be reused across multiple quizzes, creating a reusable question pool. After completing a quiz, the user receives an evaluation of their answers.

3. Methods and Tools

The following technologies were used for the implementation:

  • Python / Django - Backend, routing, and database access
  • Django ORM - Modeling the database and relationships between quizzes, questions, and answers
  • HTML / CSS - Structure and styling of the user interface
