Big O-Notation

It is used to compare algorithms. The comparison focuses on time and space complexity, which refers to the number of required computational operations and the memory usage.

Python-examples of different time complexities:

O(n)

def print_items(n):
    for i in range(n):
        print(i)

print_items(10)

Drop Constants

O(2n) → O(n) 

def print_items(n):
    for i in range(n):
        print(i)

    for i in range(n):
        print(i)

print_items(10)

O(n2)

def print_items(n):
    for i in range(n):
        for j in range(n):
            print(i, j)

print_items(10)

Drop Non-Dominants

O(n2+n) → O(n2) 

def print_items(n):
    for i in range(n):
        for j in range(n):
            print(i, j)

    for i in range(n):
        print(i)

print_items(10)

O(1)

def add_items(n):
    return n + n

print(add_items(10))

O(log n)

An example of this would be binary search. At each step, the search range is halved.

Different Terms for Inputs

O(a+b) 

def print_items(a, b):
    for i in range(a):
        print(i)
    
    for j in range(b):
        print(j)

print_items(3, 5)

Listen

my_list.append(15) and my_list.pop() have a time complexity of O(1). my_list.insert(5, 14) has a time complexity of O(n).
